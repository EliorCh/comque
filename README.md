# אתר תרגול למבחן · מבוא לתקשורת מחשבים (141418)

אתר HTML עצמאי לתרגול לפני המבחן בקורס "מבוא לתקשורת מחשבים" של המכללה האקדמית תל אביב-יפו.

## הפעלה

פתחו את `index.html` בדפדפן. אין צורך בשרת — האתר עובד מקומית מהקובץ.

## תוכן

- **95 שאלות** מ-8 מבחנים אמיתיים (2023–2025) + שאלות חיזוק מהסיכום הרשמי. לשאלות חישוב/הצפנה נבחרות יש גרסאות עם נתונים משתנים (variants).
- כל שאלה כוללת הסבר מפורט לתשובה הנכונה ולשגויות
- שמירה אוטומטית של היסטוריית מבחנים ושאלות מסומנות (ב-localStorage של הדפדפן)

## מצבי תרגול

1. **סימולציית מבחן מלא** — 20 שאלות אקראיות, טיימר של 90 דקות, ניווט בין שאלות וחישוב ציון בסוף
2. **תרגול לפי נושא** — סינון לפי שכבת רשת או נושא ספציפי, עם הסברים אחרי כל שאלה
3. **גלישה במאגר** — צפייה בכל השאלות, מסודרות לפי מבחן
4. **שאלות מסומנות** — שאלות שסימנתם לחזרה וטעויות אחרונות

## נושאים

- **שכבת אפליקציה** — HTTP, DNS, Cookies, Proxy
- **שכבת תעבורה** — TCP, UDP, rdt, Stop & Wait, GBN
- **שכבת רשת** — IP, NAT, DHCP, ARP, Routing (DV/LS/RIP/OSPF/BGP)
- **שכבת ערוץ** — ALOHA, CSMA/CD, Ethernet, Switch/Hub/Bridge
- **אבטחה** — RSA, MAC, Nonce, DOS
- **כללי** — השהיות, מודל שכבות, חישובים מספריים

## הערות

- חלק מהתשובות סומנו כ"לאימות" — מומלץ לוודא מול ההרצאות והסיכום הרשמי
- האתר עובד אופליין לאחר טעינה ראשונה (פונטים ו-KaTeX נטענים מ-CDN)
- היסטוריית התרגולים נשמרת רק בדפדפן הספציפי — לא חוצה מכשירים

## קבצים

```
index.html       — עמוד ראשי (נקודת כניסה)
styles.css       — עיצוב
app.js           — לוגיקת האפליקציה
questions.js     — מאגר השאלות
```
